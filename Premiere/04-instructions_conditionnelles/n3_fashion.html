Instructions conditionnelles en Python

Es-tu un fashion victim ?

Pour être à la mode, le pantalon doit être blanc ou noir et la chemise d'une couleur différente de celle du pantalon. Complétez le programme pour qu'il affiche le message "Vous êtes une fashion victim", ou bien "Vous avez besoin d'un reloocking". 


		 
		 
	

	

		

		

	

	
# Solution de l'exercice

couleurPantalon = input("couleurPantalon ? ")
couleurChemise  = input("couleurChemise  ? ")

if (couleurPantalon == "blanc" or couleurPantalon == "noir") and ( not (couleurChemise == couleurPantalon )):
   reponse = "Vous êtes une fashion victim"
else:
   reponse = "Vous avez besoin d'un reloocking"
	
print(reponse)
