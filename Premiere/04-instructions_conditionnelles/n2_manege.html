Instructions conditionnelles en Python

Est-ce que je peux monter sur le manège ?

Pour monter sur le manège, il faut avoir plus de 11 ans et peser au moins 45 kg. Complétez le programme pour qu'il affiche le message "Manège autorisé", ou bien "vous êtes trop jeune" ou "vous n'êtes pas assez lourd". 


		 
		 
	

	

		

		

	

	
# Solution de l'exercice

age = int( input("age ? ") )
poids = float( input("poids ? ") )

if age > 11 and poids >= 45:
   reponse = "Manège autorisé"
elif age <= 11 and poids >= 45:
   reponse = "vous êtes trop jeune"
else:
	reponse = "vous n'êtes pas assez lourd"
	
print(reponse)
