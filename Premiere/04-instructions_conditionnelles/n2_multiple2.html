Instructions conditionnelles en Python

multiple de 5 et pair ?

Complétez le programme pour qu'il affiche le message "ce nombre est pair et multiple de 5", "ce nombre est impair et multiple de 5", "ce nombre est pair mais pas multiple de 5" ou "ce nombre est impair mais pas multiple de 5", 


		 
		 
	

	

		

		

	

	
# Solution de l'exercice

x = float( input("x ? ") )

if x%2 == 0 and x%5 == 0:
    reponse = "ce nombre est pair et multiple de 5"
elif x%2 != 0 and x%5 == 0:
	reponse = "ce nombre est impair et multiple de 5"
elif x%2 == 0 and x%5 != 0:
    reponse = "ce nombre est pair mais pas multiple de 5"
else:
	reponse = "ce nombre est impair mais pas multiple de 5"
	
print(reponse)
