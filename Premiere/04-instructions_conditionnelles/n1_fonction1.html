Instructions conditionnelles en Python

Fonction définie par partie

Complétez le programme pour qu'il affiche :
  • \(x\) si \(x < 0\)
  • \(x^2 - 1\) si \(0 \leqslant x < 1\)
  • \(x + 5\) si \(x \geqslant 1\) 


		 
		 
	

	

		

		

	

	
# Solution de l'exercice

x = float( input("x ? ") )

if x < 0:
    y = x
elif 0 <= x < 1:
    y = x**2 - 1
else:
	y = x + 5
	
print("y = ", y)
