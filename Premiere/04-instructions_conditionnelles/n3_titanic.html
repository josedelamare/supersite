Quelle chance de survie pour un passager du Titanic ?

Le but de cet exercice consiste à créer un modèle simpliste de prédiction de la chance de survie d’un passager du Titanic suite à son naufrage. La survie d'un passager dépend de 5 critères :
  • le genre du passager : 0 pour homme et 1 pour femme ;
  • l’âge du passager : un entier positif ;
  • la classe du voyage du passager : 1, 2 ou 3 ;
  • le prix du ticket du voyage en £ : un flottant ;
  • le numéro de cabine : un entier entre 0 et 762.
On considère qu’un passager a plus de chances de survivre si :
  1. c’est une femme ;
  2. il est mineur (<18 ans) ou sénior (≥60 ans) ;
  3. il voyage en classe 1 ou 2 ;
  4. le prix du ticket est au moins de 10, 35 £ (inclus) ;
  5. son numéro de cabine est pair et multiple de sept.
On veut déterminer à quel point un passage correspond aux cinq critères. Il faut tester tous les critères. S’ils sont tous vérifiés, il faut afficher la chaîne de caractères "Très probable". Si seulement trois ou quatre critères sont vérifiés, il faut afficher "Probable". Si aucun critère n’est vérifié il faut afficher "Quasiment impossible" et dans les autres cas, il faut afficher "Peu probable". Complétez le programme pour qu'il affiche la probabilité de survie du passager. 


		 
		 
	

	

		

		

	

	
# Solution de l'exercice

sexe = int(input("quel est votre sexe ? 0 pour un homme et 1 pour une femme"))
age = int( input("quel est votre âge ? "))
classe = int(input("dans quelle classe voyagez-vous ?"))
tarif = float(input("combien avez-vous payé votre billet ?"))
cabine = int(input("quel est le numéro de votre cabine ?"))

criteres = 0

if sexe == 1:
	criteres = criteres + 1 
if age >= 60 or age < 18:
	criteres = criteres + 1 
if classe == 1 or classe == 2:
	criteres = criteres + 1 
if tarif >= 10.35:
	criteres = criteres + 1 
if cabine%2 == 0 and cabine%7 == 0:
	criteres = criteres + 1 
	
if criteres == 5:
	probabilite = "Très probable"
elif criteres == 3 or criteres == 4:
	probabilite = "Probable"
elif criteres == 0:
	probabilite = "Quasiment impossible"
else:
	probabilite = "Peu probable"
				
print(probabilite)
