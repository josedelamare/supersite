Instructions conditionnelles en Python

Tirage de photos

Un magasin de photos propose le développement au tarif de 0,16 € l’unité. Le tarif devient 0,12 € l’unité pour une commande d’au moins 75 photos. Complétez le programme pour qu'il affiche le prix à payer suivant le nombre de photos à tirer. 


		 
		 
	

	

		

		

	

	
# Solution de l'exercice

nombre = int( input("combien de photos à tirer ? ") )

if nombre >= 75:
    prix = nombre * 0.12
else:
	prix = nombre * 0.16
				
print(prix)
