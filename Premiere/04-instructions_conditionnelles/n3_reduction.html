Instructions conditionnelles en Python

Une réduction bienvenue !

Corriger le programme suivant pour qu'une réduction de 5 % soit appliquée à un prix atteignant ou dépassant les 100 €. 


		 
		 
	

	

		

		

	

	
# Solution de l'exercice

prix = float( input("quel est le prix ? ") )

if prix >= 100:
    prix = prix - 5/100 * prix

print("le prix réduit est :", prix)
